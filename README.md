ABC Health Care service is a popular company in which they provide medical
laboratory services (Medical tests) directed by the doctors. The Company management has
decided to develop their service by implementing an Information system so that to deliver the
test results quickly and easily through the system for both doctors and the patients. Other
requirement for the desired system are given below
Requirements
 There will be 3 types of users
1. Registered Doctors- they should be able to view test results of the patients who
directed by them only.
2. Medical lab technologists- they should have all the facilities such as view, delete,
update, and add test results.
3. Cashier – they should be able to view results of any patients as well as they can
only add patient personal information, test id, test name and invoice id
 Need to store payment information and 5% commission of the payment is granted to
the doctor who directed the patient.
 A text message should be sent to inform the patient when the test report is completed
and the test report should be emailed.
 A test result should include following details
 Invoice No
 Technologist ID
 Cashier ID
 Patient Name
 Address
 Name of the directed Doctor
 Received Date
 Issued Date
 Result
 The System should be able to get following reports
 Daily income
 Commissions given for each doctor monthly
 Medical tests information done by each Medical lab technologist monthly
 Medical tests information remaining to be issued at the time of looking for
