# ABC Medical Test Company System 🏥🔬

## 📌 Project Overview
ABC Health Care service provides medical laboratory services and medical tests directed by doctors. This system is designed to streamline and manage the process of recording, viewing, and delivering medical test results efficiently for doctors, patients, and laboratory staff. 

It handles patient information, test results, doctor commissions, automated notifications, and comprehensive reporting.

## 🚀 Features
The system contains robust functionalities categorized by different user roles and management modules:

* **User Management:** Admin controls for user registration, authentication, updating, and deletion.
* **Patient Information Management:** Cashiers add/update patient info, while Doctors and Cashiers can view it.
* **Test Management:** Lab technologists add, update, delete, and process test results. Cashiers and Doctors can view the status and results.
* **Payment & Commission Management:** Stores payment information and calculates a 5% commission for the doctor who directed the patient.
* **Automated Notification System:** * Sends SMS to patients when test reports are completed.
    * Emails test reports directly to patients.
* **Reporting System:** Generates critical analytical reports:
    * Daily Income Reports.
    * Monthly Doctor Commission Reports.
    * Monthly Test Analysis Reports (tests done per technologist, pending tests to be issued).

## 👥 User Roles & Permissions
The system operates with three main types of authenticated users:

1.  **Registered Doctors:** Can exclusively view the test results of the patients who were directed by them.
2.  **Medical Lab Technologists:** Have full access to test result management (Add, View, Update, Delete) and handle the SMS/Email notification logs.
3.  **Cashier:** Can view results of any patient, add patient personal information, input test IDs, test names, invoice IDs, and manage payment information.

## 📄 Test Result Structure
Each test result generated by the system includes:
* Invoice No
* Technologist ID & Cashier ID
* Patient Name & Address
* Name of the directed Doctor
* Received Date & Issued Date
* Final Result
